## ✨ Overview

This project uses **Machine Learning (Logistic Regression)** to predict whether a person will buy insurance based on their **age**.

> 🎯 Goal: Classify customers into:
- 🟢 1 → Will buy insurance  
- 🔴 0 → Will NOT buy insurance  

---

## 📊 Dataset Information

| Feature | Description |
|--------|-------------|
| 🎂 age | Age of the customer |
| 🛡️ bought_insurance | Target (0 or 1) |
